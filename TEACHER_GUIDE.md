# 결보강 관리 시스템 - 교사용 사용 설명서

수업 교체·보강 신청과 처리를 위한 간단한 사용법입니다.

## 1. 접속하기

관리자 선생님이 안내한 주소로 접속합니다. (예: `http://13.124.173.16` 또는 학교 도메인)

### 최초 로그인

- **ID**: 본인 성함 (시간표에 등록된 이름 그대로)
- **초기 비밀번호**: `1234` (학교 관리자 안내 참고, 변경되었을 수 있음)

처음 로그인하면 **비밀번호를 반드시 새로 바꿔야** 다음 화면으로 넘어갑니다.

- 현재 비밀번호: `1234`
- 새 비밀번호: 본인이 정한 비밀번호로 입력 후 "변경 후 시작하기"

> 비밀번호를 잊어버리면 본인이 다시 찾을 방법이 없습니다. 관리자 선생님께 "비밀번호 초기화"를 요청하면 다시 `1234`로 돌아갑니다.

## 2. 화면 둘러보기

로그인하면 왼쪽에 메뉴가 있습니다.

| 메뉴 | 하는 일 |
|---|---|
| 내 시간표 | 이번 주 내 수업, 교체·보강 반영 결과 확인 |
| 교체 신청 | 내 수업과 다른 교사의 수업을 맞바꾸는 신청 |
| 보강 신청 | 내 수업을 다른 교사가 대신 맡아주도록 요청 |
| 교체·보강 현황 | 내가 보내거나 받은 요청의 진행 상태 확인, 취소 |
| 계획서 작성 | 결보강 계획서(엑셀/PDF) 다운로드 |
| 알림·요청함 | 새 알림 확인, 받은 요청 수락/거절 |

화면 위쪽 초록 배너에는 "받은 대기 요청", "안 읽은 알림" 개수가 항상 표시되니 참고하세요.

## 3. 수업 교체 신청하기

같은 학반 수업을 다른 요일/교시로 맞바꾸고 싶을 때 사용합니다.

1. **교체 신청** 메뉴로 이동
2. **교체 희망일** 선택
3. 그 날짜의 내 수업 목록에서 바꾸고 싶은 교시의 **"후보 찾기"** 클릭
4. 오른쪽 **"교체 후보"** 목록에서 이번 주/다음 주 중 원하는 후보를 골라 **"요청 보내기"**
5. 상대 교사가 수락하면 자동으로 시간표에 반영됩니다.

> 후보가 안 보이면: 같은 학반 수업인지, 상대 교사가 그 시간에 공강인지, 이미 다른 요청이 걸려있는 시간은 아닌지 확인해 보세요.

## 4. 보강 신청하기

내가 못 가르치는 수업을 다른 교사가 대신 맡아주길 요청할 때 사용합니다.

1. **보강 신청** 메뉴로 이동
2. **보강 날짜** 선택
3. 보강이 필요한 내 수업 옆 **"보강 후보 찾기"** 클릭
4. 오른쪽 후보 교사 목록에서 요청할 교사를 골라 **"보강 요청"**
5. 상대 교사가 수락하면 확정됩니다.

## 5. 받은 요청 확인하고 응답하기

다른 교사가 나에게 교체·보강을 요청하면:

1. **알림·요청함** 메뉴로 이동 (사이드바에 안 읽은 알림 개수가 표시됩니다)
2. **"받은 요청"** 목록에서 내용 확인 (날짜, 교시, 상대 교사, 수업 내용)
3. 괜찮으면 **"수락"**, 어려우면 **"거절"**

수락하면 즉시 시간표에 반영됩니다. 거절해도 불이익은 없으니 어려운 요청은 편하게 거절하세요.

## 6. 신청 취소하기

내가 보낸 요청을 취소하거나, 이미 확정된 교체·보강을 되돌리고 싶을 때:

1. **교체·보강 현황** 메뉴로 이동
2. 해당 항목에서 **취소** 또는 **확인 후 삭제**

확정된 건은 취소하면 시간표가 원래대로 돌아갑니다.

## 7. 계획서 다운로드 (필요한 경우)

행사·출장 등으로 결보강 계획서 제출이 필요할 때:

1. **계획서 작성** 메뉴로 이동
2. 기준 날짜 선택 → 해당 주 내역 확인
3. **엑셀 내려받기** 또는 **PDF 내려받기**

일반 교사는 본인과 관련된 내역만 보입니다.

## 8. 자주 묻는 질문

**Q. 교체·보강 후보가 하나도 안 보여요.**

A. 다음을 확인해 보세요.

- 같은 학반 수업을 고르셨는지
- "이번 주"/"다음 주" 탭을 잘못 보고 있는 건 아닌지
- 상대 교사가 그 시간에 이미 수업이나 순회 중은 아닌지
- 신청 마감 시간이 지난 건 아닌지 (화면에 안내 문구가 뜹니다)

**Q. 비밀번호를 잊어버렸어요.**

A. 스스로 찾을 방법은 없고, 관리자 선생님께 "회원·교사 관리"에서 비밀번호 초기화를 요청하면 다시 `1234`로 초기화됩니다. 그 뒤 다시 로그인해서 새 비밀번호로 바꾸시면 됩니다.

**Q. 수락했는데 시간표에 안 보여요.**

A. 화면 오른쪽 위 **"새로고침"** 버튼을 눌러보세요. 그래도 안 보이면 관리자 선생님께 문의해 주세요.

**Q. 신청한 요청에 아무 응답이 없어요.**

A. 요청마다 응답 마감 시간이 있습니다. 마감이 지나면 자동으로 만료되고, 다른 후보에게 다시 신청하시면 됩니다.
